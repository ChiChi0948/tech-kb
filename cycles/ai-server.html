AI 伺服器

2023～ · 生成式 AI 訓練與推論驅動的科技週期。核心是「大量 GPU 算力 + 高頻寬記憶體 + 先進封裝把它們塞在一起」,而真正定義這個週期的不是 GPU 本身,而是圍繞 GPU 建構的整套硬體帳本——這頁就是要把這本帳拆開。

週期概述

AI 伺服器週期的核心需求是「算力」:訓練大型語言模型需要大量 GPU,而 GPU 要發揮效能,得搭配高頻寬記憶體 HBM與先進封裝 CoWoS把它們緊密整合。這個週期由 輝達 幾乎定義了規格——GPU 淨利率約 55%,一個 Oracle 的 400 億美元訂單就能貢獻超過 200 億美元純利潤。

但把眼光只放在 GPU 上會錯過整條產業鏈。真正決定產業結構的是圍繞 GPU 展開的九大硬體賽道——GPU 只是最大宗、最貴的一塊,不是唯一的一塊。

賽道一:算力晶片與封裝 — 70% 佔比,但競爭生態複雜

1. GPU 訓練與推論晶片(絕對核心)

輝達是絕對霸主。GPU 淨利率約 55%,一個 Oracle 400 億訂單就能貢獻超過 200 億純利潤。CUDA 軟體生態鎖定開發者+晶片架構領先,客戶轉換成本極高。AMD 與 Intel 在硬體性能上力追,但要真正撼動輝達仍極其艱難。

2. 大廠自研 ASIC(結構性威脅)

為了擺脫對輝達的依賴,雲端大廠都在自研專用 AI 晶片:Google TPU、Amazon Trainium、Meta MTIA。實測 Google TPU 的 MLCC 用量比預期還高——這代表 ASIC 不僅是 rumor,已經在真實出貨,並牽動整條材料供應鏈。

3. 高頻寬記憶體 HBM

幾乎和 GPU 綁定的黃金搭檔。目前市場由 SK 海力士、三星、美光壟斷,而且進入 HBM4 世代升級週期——每一代升級,單價、單卡用量、良率要求都上一個台階。

4. 先進封裝 CoWoS

台積電獨佔的技術,一路吃盡輝達與 AMD 全部訂單。這是整個 AI 供應鏈的最核心瓶頸——設計再多、晶片再多,沒 CoWoS 產能就組不出成品。

賽道二:網路與互聯 — 15-20%,通信需求跳躍式暴漲

底層邏輯:當 GPU 規模從幾百顆擴展到幾十萬顆,晶片之間的通信需求不是線性增加,而是跳躍式暴漲——這才是「網路才是新的算力」的真正含義。

1. 光模組(數量最多、量價齊升)

每一顆 GPU 都需要多個光模組完成互聯,傳輸速率從 400G → 800G → 1.6T → 3.2T 一路狂奔。全球出貨完全由中國廠商主導:中際旭創、天孚通信、新易盛。

2. 高速交換機(架構定義者)

整個 AI 集群的「大腦」。輝達力推 NVLink 生態把交換機也自產化;博通用其 Tomahawk 交換晶片,以更開放的姿態與輝達正面對抗。這是一場關於 AI 集群未來架構的戰爭。

3. 銅纜互聯(短距新戰場)

機櫃內部短距離互聯用銅纜(便宜、可靠、功耗低);跨機櫃長距離互聯用光模組。銅纜技術含量看似不高,但涉及大量精密製造與高頻連接器,由安費諾、立訊精密、兆龍互連等頭部連接器廠商壟斷。

4. 算力線路板(CCL 材料軍備競賽)

信號速率飆升到 224G、甚至下一代 448G,對板材的介電損耗要求跳躍式提高。「M8 等級」以上 CCL 材料成為 AI 主板剛需,由 台光電 為首的少數頂級 CCL 廠寡佔。詳見 CCL 銅箔基板。

賽道三:電氣與散熱 — 10%,但整個電力鏈正在重構

底層趨勢:單個機櫃功率從 100kW 一路衝到 600kW、1MW+。這催生兩個結構性變化:液冷全面取代風冷、整個資料中心供電架構往800V 高壓直流重構。

1. 液冷散熱(從 optional 變 mandatory)

傳統風冷完全帶不動 100kW+ 機櫃。液冷不再是選項,而是唯一選項——冷板式、浸沒式方案齊上陣。雙良節能、英維克、Vertiv、施耐德 這些老牌工業與熱管理龍頭直接吃到二次騰飛。

2. 電力供應與 UPS(高壓直流化)

800V DC 從無到有、規模鋪開。施耐德、日立能源、伊頓等傳統電力巨頭獲得結構性訂單。

3. 功率半導體 — 資料中心的隱形受益者

從電網 kV 級一路降到 GPU 核心 1V 級,每一階降壓都需要 MOSFET / IGBT / SiC / GaN。Infineon 產品線最完整、覆蓋 SST 到 GPU 周邊全電壓段;STM 強在 800V 高壓直電與中段轉換。過去被歸類為「車用+工業」題材,現在正在被市場重新理解為 AI 基建鏈的核心一環。

4. 被動元件(MLCC / 電解電容大爆發)

Rubin 一台主板 25,000 顆 MLCC、GB200 3 萬顆、整機櫃可達 44 萬顆——是傳統伺服器的 8 倍且單價更高。48V 諧振電容從邊緣變主流,由 禾伸堂直接受惠;Commodity 漲價由 國巨、華新科承接;PSU 牛角電解電容全球包廠搶貨。

投資邏輯 — 誰是真正的「賣鏟人」?

幾千億美元 CapEx 流下來,但錢的流向極不均勻。整條產業鏈可以分成三類:

類型特徵與代表
絕對王者護城河極深技術、專利、生態雙壁壘。錢一定流過,而且大部分都是它們吃。代表:輝達(GPU)、台積電(CoWoS)、SK 海力士(HBM)、Infineon(功率半導體)、村田(MLCC)、施耐德(電力設備)
賣鏟人位卡關鍵環節不是最耀眼、但被產業鏈強行推動升級的關鍵環節供應商。代表:台光電(CCL M8)、中際旭創(光模組)、禾伸堂(48V 諧振)、雙良(液冷)、STM(800V 直電)
紅海競爭彈性最大但淘汰快技術門檻相對低、產能可快速擴張。代表:PCB 中低階、部分連接器、通路商、部分陸系材料商

核心投資邏輯:把握 AI 大基建浪潮的核心,不是追 GPU 這種眾所週知的英雄,而是找到那些位在關鍵環節、被產業鏈強行推動升級的「賣鏟人」。它們可能沒有輝達那麼耀眼,但它們是這場淘金熱裡真正能穩定收錢的人。

與其他週期對照

同一批代工廠與封測廠(台積電、日月光)在不同週期角色不變,但訂單重心轉移。手機週期訂單是 SoC + 影像感測 + 面板;AI 伺服器週期訂單重心是 GPU + HBM + CoWoS + 高階 CCL + 800V 功率半導體。這正是「同一批名字、不同週期扮演不同角色」的經典範例。

電動車週期也和 AI 週期存在交叉——例如功率半導體、車用 MCU 都是兩者共用資產。資料中心週期則直接和 AI 週期融合,已幾乎不可分。
