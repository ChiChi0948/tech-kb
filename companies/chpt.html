公司

精測 CHPT

Load Board 龍頭 + 聯發科手機 SoC 探針卡幾乎獨家。Nvidia GPU Load Board 從 Blackwell 世代 25% 大幅躍升至 Rubin 世代 70%——但 Google Humming Fish CP 板掉給雍智、手機下修、MCC Burn-in 爐子有問題,是短期壓力點。

基本定位

營收結構

  • 探針卡 PCB:68%
  • Load Board:21%
  • 其他測試板:11%

與同業差異

精測的護城河在「板子」——探針卡的 PCB 基板、FT/SLT 的 Load Board、Burn-in Board。他不做針(針另外向 FormFactor 買),但整合成套探針卡出貨給客戶。這讓他在 Load Board 端幾乎沒有台廠對手,同時卡到聯發科整套探針卡供應。

一句話定位 · Load Board 龍頭 + 聯發科手機 SoC 探針卡穩定基本盤 + Nvidia GPU Rubin 世代 Load Board 大躍升 + Humming Fish CP 掉單短期陣痛。

核心客戶

Nvidia GPU · Load Board 的關鍵躍升

Blackwell 世代Load Board 25%
Rubin 世代Load Board 70% · 對手 RDNova 遇到瓶頸,精測拿下大單
Nvidia 電競 GPULoad Board 100%(旺矽出板都是精測做)

聯發科 · 穩定基本盤

  • 手機 SoC(旗艦 / 天璣)探針卡 100%——整套探針卡都是精測
  • 每年穩定貢獻精測 20% 營收

Google TPU · 過渡期陣痛

ZebraFish 世代CP 探針卡 100% · CP + FT + SLT + Burn-in 全包
Humming Fish 下一代CP 板掉給 雍智(從精測手上搶到) · 但 FT + SLT Load Board 仍是精測

其他

特斯拉 AI5探針卡 30%(旺矽 70%)、FT Load Board 100%

產品線

探針卡端

  • PCB 基板自製 + 針向 FormFactor 買 + 整合出貨 → 這是聯發科 SoC 的整套供應方式
  • Cobra 主力,不做 MEMS 針(所以在 AI GPU CP 端吃不到)

Load Board 端

  • FT Load Board:Nvidia GPU、Google TPU、特斯拉 AI5 全供
  • SLT Load Board:與 FT 同客戶,SLT 用量 4–5 倍放大
  • Burn-in Board:Google TPU 供應(MCC 爐子問題可能影響)

成長動能

1. Nvidia Rubin Load Board 25% → 70%

這是精測今年最大的正面驚喜。對手 RDNova 在 Rubin 世代技術上遇到瓶頸,精測拿下大單。Rubin 的測試時間、pin count、ASP 都比 Blackwell 高,乘上份額躍升是「量價齊揚」。

2. SLT Load Board 加碼

SLT 用量是 FT 的 4–5 倍,精測在 Nvidia、Google、特斯拉的 FT Load Board 100% 通吃,延伸到 SLT 就是同一比例放大 4–5 倍。

3. 聯發科手機 SoC 每年 20% 營收穩定

雖然手機今年下修,但聯發科旗艦系列的探針卡整套供應仍是精測基本盤。

壓力點

三個短期壓力 · (1) 手機下修拖累聯發科 SoC 出貨、(2) Humming Fish CP 板掉給雍智、(3) MCC Burn-in 爐子有良率問題,Humming Fish 若 MCC 出狀況會轉京元電。

Humming Fish CP 掉單的意義

ZebraFish 世代精測 CP 板 100%,Humming Fish 掉給雍智代表 Google 願意分散供應。但 FT/SLT Load Board 仍是精測——所以不是失去客戶,而是失去「CP 這一站」的獨佔。

估值

  • 2027 EPS × 40 倍(較旺矽穎崴的 50 倍低,反映短期壓力)
市場評價 · 精測估值折價反映三個短期壓力,若 Rubin Load Board 70% 順利放量、MCC 爐子問題解決、聯發科手機修復,估值有補漲空間。分析師節奏上把它排在旺矽、穎崴之後。
