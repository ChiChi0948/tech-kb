公司

穎崴 WinWay

台廠唯一測試座專營公司,SLT 週期最大受惠者。Nvidia + AMD + Google 三大客戶合計佔 80% 營收,SLT 用量是 FT 的 4–5 倍 直接放大他的規模。Hypersocket 2027 放量是下一步關鍵。

基本定位

營收結構

  • 測試座:50–60%
  • 探針卡:30%
  • 其他:10–20%
一句話定位 · 台廠唯一測試座專營 + Nvidia AI GPU FT/SLT 接近獨吃 + Hypersocket 新技術做出差異化 + 探針卡 MEMS 因外購拖累毛利。

核心客戶

Nvidia營收 30–40% · AI GPU FT Socket 接近 100%、今年切入 SLT 供應鏈、Mellanox 探針卡 70–80%、電競 GPU 探針卡 50%
AMD營收 20–30% · Server CPU SLT Socket 主力 · 今年需求已上修兩次,Server CPU 缺貨
Google營收 20–30% · TPU FT + SLT 都做,SLT 與 Smith 分

三大客戶合計約 80% 營收,結構性受惠 AI 資料中心投資。

探針卡端(只服務 Nvidia)

  • AI GPU MEMS 80%
  • Mellanox 網通 70–80%
  • 電競 GPU 50%(另一半是旺矽)

產品線

測試座三條線

Coaxial Socket目前主流 · 訊號乾淨、可測幾十 GHz
Hypersocket穎崴獨家 · 有散熱功能 + 換針快 · ASP 比 Coaxial 貴 20–30%
Burn-in SocketRubin 世代開始導入

探針卡:設計自己、針向 Technoprobe 外購

穎崴不生產 MEMS 針,而是向意大利 Technoprobe(全球市佔第一)採購,自己做板子上針、賣給客戶。這帶來重要的毛利率結構——見下方毛利率壓力章節。

成長動能

1. Nvidia SLT 切入

穎崴原本只做 Nvidia AI GPU 的 FT Socket,今年首度切入 SLT 供應鏈。SLT 用量是 FT 的 4–5 倍,意味著 Nvidia GPU 這一站的 Socket 收入直接倍增。

2. AMD SLT 主力

AMD Server CPU 缺貨、今年需求上修兩次,穎崴 SLT Socket 是主要供應。這一塊是意外驚喜。

3. Hypersocket 2027 放量

  • 2026 小量工程樣品(客戶想用但穎崴產能不夠)
  • 2027 正式放量(Nvidia + AMD)
  • ASP 比 Coaxial 高 20–30% · 拉毛利率

4. Google TPU 兩世代都吃

ZebraFish 現在、Humming Fish 下一代,SLT 都是穎崴 + Smith 分,穩定基本盤。

SLT 用量放大的量化 · 例如 Rubin 需要 1,000 顆 FT Socket,就要 4,000–5,000 顆 SLT Socket。穎崴今年 SLT 切入 Nvidia,是他今年 EPS 上修的核心原因。詳見 測試座 · SLT 用量放大原理。

毛利率壓力

今年的矛盾 · 營收大成長,但毛利率反而衰退——原因是上半年出很多 MEMS 探針卡(給 AMD CPU + Mellanox),而 MEMS 針向 Technoprobe 外購,毛利率只有 30–40%,拖低整體。

穎崴 vs 旺矽的商業模式對比

這是理解兩家股價節奏差異的關鍵。詳見 探針卡 · MEMS 自製 vs 外購。

  • 旺矽:MEMS 自製 → MEMS 越多毛利越好
  • 穎崴:MEMS 外購 → MEMS 越多毛利被壓越嚴重(短期矛盾),必須靠 Hypersocket 拉回

估值

  • 2028 EPS × 50 倍
