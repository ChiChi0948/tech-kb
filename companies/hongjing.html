公司

鴻勁 Hongjing

台廠唯一 SLT Handler 主力供應商。ATC 主動溫控讓 Handler ASP 從 NT$6M 跳到 NT$20M+(漲 3 倍)——結合 SLT 用量放大 4–5 倍,是本波 AI 測試設備最具槓桿效果的成長故事。

基本定位

鴻勁專注在 Handler 分選機——晶片從輸送帶進來後,機械手臂抓起、對位、施壓、分類的機台。他不是最大的 Handler 廠(海外有 Cohu、Advantest),但在 SLT Handler 這個新戰場、加上 ATC 主動溫控這個 AI 晶片專屬需求,鴻勁是台廠唯一的主力。

一句話定位 · SLT Handler 唯一台廠主供 + ATC 溫控讓 ASP 三倍跳 + SLT 用量放大 4-5 倍 + CPO Insertion 4 的系統級測試主玩家。雙重放大效應驚人。

核心產品:SLT Handler + ATC

什麼是 ATC

ATC = Active Temperature Control(主動溫控)。Rubin 吃 1,800W 自己會發熱,若要測「正常運作溫度」下的表現,就要主動把熱抽走(冷板)或加溫(低溫測試)。傳統無主動溫控的 Handler 根本無法應付。

ASP 三倍跳

傳統 HandlerNT$ 6M / 台
含 ATC 的 SLT HandlerNT$ 20M+ / 台(漲 3 倍)

客戶端

Nvidia AI GPU、Broadcom ASIC 都用鴻勁的 SLT Handler。這與 穎崴、精測 的客戶重疊,形成台廠「後段 SLT 測試」的完整套裝。

成長動能

雙重放大效應

這是鴻勁投資邏輯的精髓——兩個放大同時發生:

  • 用量放大:SLT Handler 用量是 FT Handler 的 4–5 倍(SLT 時間長 20–30 倍)
  • ASP 放大:ATC 溫控讓每台 Handler 從 NT$6M 跳到 NT$20M+

兩者乘起來就是每一顆 AI GPU 對應的 Handler 收入放大近 15 倍——這在台灣測試設備廠裡是最具槓桿效果的一家。

量化直覺 · 一顆 Rubin GPU 對應到鴻勁的收入,大概是同一顆 Hopper 時代對應到傳統 Handler 廠收入的 10–15 倍。這是「同一顆晶片」的收入放大,還沒算 Nvidia 出貨量本身的成長。

CPO Insertion 4

CPO(Co-Packaged Optics)測試分四階段,鴻勁在最後一站 Insertion 4:CPO Module + System Level Test 卡到台廠主要玩家的位置。這是 SLT Handler 的自然延伸——CPO 模組還是要用類似的機械手臂搬運、加壓、溫控,鴻勁的技術可以延伸過去。

詳見 測試座 · CPO 四階段 Insertion。

競爭

SLT Handler鴻勁台廠主力 · 對手主要是致茂
Prober(CP 端)TEL、Tokyo Seimitsu 日廠壟斷,鴻勁不做

為什麼鴻勁能在 SLT Handler 卡到位

SLT Handler 是相對新的市場,傳統 FT Handler 大廠(Cohu、Advantest)反應速度沒有台廠快,加上 ATC 是需要客製化整合的技術,鴻勁貼近台廠客戶(京元電、日月光)有優勢。

風險 · 致茂國際同樣在做 SLT Handler,是鴻勁最直接的台廠對手。海外 Cohu 若切入 SLT 市場也會分食。但短期內 ATC 這道門檻讓鴻勁維持領先。
